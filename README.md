# Скрипты установки modern-g7-32

Скрипты для установки пакета modern-g7-32 для Typst.

## Использование

### Windows (PowerShell)
```powershell
.\install.ps1
```

### Linux/macOS (Bash)
```bash
./install.sh
```

Скрипты автоматически скачают и установят последнюю версию пакета modern-g7-32 в соответствующую директорию пакетов Typst.
